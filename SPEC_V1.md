# SPEC v1

## Core Value
Build a small English resume-template site that converts adjacent no-experience job-search traffic into a stronger machine-learning-resume path.

## User Stories
- Role: fresh graduate or no-experience candidate
- Action: land on a resume page that matches their current search intent
- Result: copy useful resume structure, improve project bullets, and move toward the ML-resume pillar page

## Scope
- Convert 3 existing markdown page drafts into a reusable site structure
- Add page metadata, shared layout rules, internal linking rules, and homepage/index shell
- Keep content source in markdown for now; no deployment automation in this round

## Constraints
- Do not reopen discarded AI-title keyword paths
- Keep primary keyword language aligned with the locked brief
- Prefer simple static-site structure over premature framework complexity
- Current round goal is site-ready structure, not full production polish

## Non-Goals
- Full deployment
- Analytics wiring
- Programmatic SEO scale-out
- Rewriting the keyword strategy from scratch
